## 引言
> ### AI深度学习的定义：
> 层次化的概念让计算机构建比较简单的概念来学习复杂概念。如果绘制出表示这些概念如何建立在彼此之上的图，我们将得到一张“深”（层次更多）的图
用简答的概念去组合成复杂的概念，最后形成一个多层次的图，像是字母abc,最为简答的组成单词，由单词之间的联系组成了句子，便有了语义，用于了沟通。
